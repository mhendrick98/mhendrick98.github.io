# mhendrick98.github.io

Personal site for Michael Hendrick. Plain HTML and CSS, deployed via GitHub Pages at [www.mhendrick.com](https://www.mhendrick.com).

## Structure

- `index.html` — home page
- `projects.html`, `research.html`, `workshops.html`, `blog.html` — content pages
- `style.css` — shared stylesheet
- `assets/` — resume and research PDFs

## Local preview

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000
